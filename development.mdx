Webhook Implementation
Setting Up Webhooks
pythonCopyfrom flask import Flask, request
from webhook_handler import WebhookHandler

app = Flask(__name__)
webhook_handler = WebhookHandler()

@app.route('/webhook', methods=['POST'])
def handle_webhook():
    event = request.json
    return webhook_handler.process(event)
Event Types
Our platform supports the following webhook events:

employee.created
employee.updated
attendance.marked
leave.requested
payroll.processed

Troubleshooting
<AccordionGroup>
  <Accordion title="Webhook verification failed">
    Check if your `WEBHOOK_SECRET` matches the one in your developer portal.
  </Accordion>
  <Accordion title="API authentication error">
    Verify your APP_ID and APP_SECRET are correctly set in your environment.
  </Accordion>
</AccordionGroup>
