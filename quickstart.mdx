Getting Started
Learn how to quickly set up and integrate with our HRMS platform using our Python template.
Initial Setup
<AccordionGroup>
  <Accordion icon="github" title="Clone the template repository">
    Get started by cloning our Python app template repository:
    ```bash
    git clone https://github.com/Aqary-Org/aqaryone-app-template.git
    cd aqaryone-app-template
    ```
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Set up your environment">
    1. Create and activate a virtual environment:
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
    ```
    2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
  </Accordion>
</AccordionGroup>
Configure Your App
<AccordionGroup>
<Accordion icon="gear" title="Environment setup">
  Copy the `.env.example` file to `.env` and update the following variables:
  ```
  APP_ID=your_app_id
  APP_SECRET=your_app_secret
  WEBHOOK_SECRET=your_webhook_secret
  ```
</Accordion>
<Accordion icon="webhook" title="Configure webhooks">
  1. Set up your webhook endpoint in the developer portal
  2. Update the webhook URL in your app settings
  3. Implement the webhook handler using our template
</Accordion>
</AccordionGroup>
